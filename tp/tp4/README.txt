*********************
    Configuration
*********************
Change the default IP Address of the server:
	- Go to "./app/src/main/java/com/example/tp4/RequestPageActivity"
	- In the "request" function
	- Change URL "http://132.207.89.31:80/" to your needs

**********************
    Compile client
**********************
To compile the client on Nexus 9 tablet:
	- Open folder tp4 (".") in Android Studio
	- Refer to the "Configuration" section above to fit your needs
	- Press Run
	- Run on Android Nexus 9 tablet via USB or AVD
